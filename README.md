## Проект "ЧатиК"
На данном этапе реализованы следующие роуты: 
- /chat
- /login
- /signup
- /profile
- /notfound
- /servererror

Для сборки проекта используйте команду:
```sh
npm run build
```

Для поднятия сервера:
```sh
npm run start
```
